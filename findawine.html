FinDaWine Application

FinDaWine est une Application Web Progressive qui permet à l'utilisateur de rechercher un Vin présent dans la cave du commerçant par plusieurs moyens différents.
L'application est présentée sur Tablette disponible à l'utilisation pour les clients en magasin.
Le client souhaitait une charte graphique simple, épurée et légèrement premium. Un logo et une couleur principale représentant le monde viticole.
L'utilisateur peut utiliser une barre de recherche et taper directement le nom d'un vin ou filtrer par item séléctionné. La fonction principale étant de pouvoir scanner directement le code barre avec la tablette pour afficher les informations du produit.
Un back-office avec login est également présent, permettant d'ajouter, de modifier ou de supprimer un produit (accessible via internet).
